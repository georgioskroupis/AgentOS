# AgentOS Docs

AgentOS combines two layers:

- Harness engineering: make each repository readable, testable, and correctable
  for coding agents.
- Symphony-style orchestration: poll eligible work, create isolated workspaces,
  start Codex runs, record state, and hand off for human review.
